Lost in the Mountains: You got lost on your hike and need to find shelter before you freeze.  
  
?- start.

Enter commands using standard Prolog syntax.  
Available commands are:  
start.             -- to start the game.  
n.  s.  e.  w.     -- to go in that direction.  
i.     -- to see your inventory.  
take(Object).      -- to pick up an object.  
use_item(Object).      -- to use an object.  
look.              -- to look around you again.  
instructions.      -- to see this message again.  
halt.              -- to end the game and quit.
